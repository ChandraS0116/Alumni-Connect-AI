### Alumni Connect AI

A modern alumni networking platform that helps students, alumni, and mentors connect, collaborate, and grow through meaningful professional relationships.

## Overview

Alumni Connect AI is designed to bridge the gap between students and alumni by providing a centralized platform for networking, mentorship, career guidance, and community engagement.

The platform enables users to discover opportunities, build professional connections, and strengthen the alumni ecosystem.

## Installation:

Clone the repository:

```bash
git clone https://github.com/ChandraS0116/Alumni-Connect-AI.git
```

Navigate to the project:

```bash
cd Alumni-Connect-AI
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
